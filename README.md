Installation
------------

0. Ensure Prerequisites:
   a. PHP v5.3+
   b. MongoDB
   c. PHP Mongo PECL Extension (v1.2+)
1. Extract all files to web accessible directory
2. Copy config/config.yml.dist to config/config.yml and adjust parameters
3. Run ./setup
